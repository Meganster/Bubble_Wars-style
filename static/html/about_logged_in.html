Bubble Wars
About us
This game is a web version of the Auralux for two players. The game is played on the field, which is an abstract space. Single-user mode is implemented using a bot. Every player has a bubble at the beginning of the game. Any bubble occupied by the player can generate units with a certain speed. The goal of the game is to capture all the located bubbles on the map. For each bubble belonging to the player that generated a certain number of units can used to capture both enemy and neutral bubbles. With the help of units, you can also to improve the bubbles already in the possession of the player.
